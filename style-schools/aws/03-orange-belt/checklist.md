# AWS Orange Belt Checklist

This checklist outlines the intermediate concepts and tasks required to achieve your AWS Orange Belt.

## Core Concepts

- [ ] VPC and networking (subnets, route tables, internet gateways)
- [ ] Load balancing (ALB, NLB, CLB)
- [ ] Auto Scaling Groups
- [ ] RDS and database management
- [ ] S3 advanced features (lifecycle policies, versioning, replication)
- [ ] Lambda and serverless architectures
- [ ] API Gateway
- [ ] CloudFormation basics (Infrastructure as Code)
- [ ] SNS and SQS for messaging
- [ ] DynamoDB fundamentals
- [ ] Container services (ECS basics)
- [ ] Cost optimization strategies

## Practical Tasks

- [ ] Design and implement a VPC with public and private subnets
- [ ] Deploy an application behind an Application Load Balancer
- [ ] Configure Auto Scaling based on metrics
- [ ] Set up an RDS database with read replicas
- [ ] Create a serverless API with Lambda and API Gateway
- [ ] Implement S3 lifecycle policies for cost optimization
- [ ] Build a CloudFormation template for infrastructure
- [ ] Set up SNS/SQS messaging pipeline
- [ ] Deploy a containerized application to ECS
- [ ] Implement cost monitoring and alerts

## Projects

- [ ] Build a scalable, highly available web application on AWS
- [ ] Create a serverless data processing pipeline
- [ ] Deploy a microservices architecture using AWS services

Once you can confidently check off these items, you are ready for the Green Belt!
