# AWS Green Belt Checklist

This checklist outlines the advanced concepts and tasks required to achieve your AWS Green Belt.

## Core Concepts

- [ ] Advanced VPC (VPC peering, Transit Gateway, PrivateLink)
- [ ] EKS (Kubernetes on AWS)
- [ ] Advanced Lambda patterns (layers, destinations, extensions)
- [ ] Step Functions and workflow orchestration
- [ ] CloudWatch advanced features (Logs Insights, Synthetics)
- [ ] AWS Organizations and multi-account strategies
- [ ] Service Catalog and governance
- [ ] Advanced security (GuardDuty, Security Hub, Secrets Manager)
- [ ] Cost management at scale (Cost Anomaly Detection)
- [ ] Advanced CloudFormation (nested stacks, StackSets)
- [ ] EventBridge for event-driven architectures
- [ ] AppSync and GraphQL APIs

## Practical Tasks

- [ ] Design and implement multi-account AWS organization
- [ ] Deploy Kubernetes cluster with EKS
- [ ] Build complex Step Functions workflows
- [ ] Implement comprehensive security monitoring
- [ ] Create Service Catalog products
- [ ] Deploy applications across multiple accounts with StackSets
- [ ] Build event-driven architecture with EventBridge
- [ ] Implement advanced cost optimization strategies
- [ ] Design and deploy AWS Landing Zone
- [ ] Create custom CloudWatch dashboards and alarms

## Projects

- [ ] Build a complete AWS Landing Zone for an organization
- [ ] Deploy a production-grade Kubernetes platform on EKS
- [ ] Implement enterprise-wide security and compliance framework

Once you can confidently check off these items, you are ready for the Blue Belt!
