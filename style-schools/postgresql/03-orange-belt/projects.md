## Project 1: Analytics Platform Database

**Time**: 2-3 weeks  
**Difficulty**: ⭐⭐⭐ Intermediate  
**Focus**: Complex queries and window functions

### Description

Design and build a database for an analytics platform with complex reporting requirements.

### Requirements

**Schema Design**:

- [ ] Events table (millions of rows)
- [ ] Users table with demographics
- [ ] Products/features table
- [ ] Time dimension table
- [ ] Proper normalization and indexing

**Complex Queries**:

- [ ] User cohort analysis
- [ ] Retention calculations
- [ ] Funnel analysis
- [ ] Revenue trends with window functions
- [ ] Top products by segment

**Performance**:

- [ ] Queries complete in <1 second
- [ ] Proper indexes on all queries
- [ ] Materialized views for dashboards
- [ ] Query optimization with EXPLAIN

### Deliverables

- [ ] Complete schema with sample data
- [ ] 20+ analytical queries
- [ ] Performance analysis report
- [ ] Dashboard mock-up
- [ ] Documentation

### Evaluation Criteria

- **Query Complexity (35%)**: Advanced SQL techniques
- **Performance (30%)**: Optimized queries
- **Design (20%)**: Well-designed schema
- **Documentation (15%)**: Clear explanations

## Project 2: Database Optimization Challenge

**Time**: 2-3 weeks  
**Difficulty**: ⭐⭐⭐ Intermediate  
**Focus**: Performance tuning and indexing

### Description

Take a slow database and systematically optimize it through indexing, query rewriting, and schema improvements.

### Requirements

**Initial Setup**:

- [ ] Create database with performance issues
- [ ] Document baseline performance
- [ ] Identify slow queries

**Optimizations**:

- [ ] Add appropriate indexes
- [ ] Rewrite inefficient queries
- [ ] Implement query caching strategies
- [ ] Optimize table structure
- [ ] Configure PostgreSQL settings

**Measurements**:

- [ ] Before/after query times
- [ ] Index usage statistics
- [ ] Cache hit ratios
- [ ] Disk I/O reduction

### Deliverables

- [ ] Original vs optimized database
- [ ] Optimization report
- [ ] Performance benchmarks
- [ ] Best practices guide

### Evaluation Criteria

- **Improvements (40%)**: Measurable performance gains
- **Methodology (30%)**: Systematic approach
- **Documentation (20%)**: Clear explanations
- **Best Practices (10%)**: General applicability
