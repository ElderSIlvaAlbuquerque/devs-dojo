# PostgreSQL Orange Belt Checklist

This checklist outlines the intermediate concepts and tasks required to achieve your PostgreSQL Orange Belt.

## Core Concepts

- [ ] Complex queries (JOINs, subqueries, CTEs)
- [ ] Window functions (ROW_NUMBER, RANK, LAG, LEAD)
- [ ] Indexing strategies (B-tree, Hash, GiST, GIN)
- [ ] Query optimization and EXPLAIN ANALYZE
- [ ] Transactions and isolation levels
- [ ] Views and materialized views
- [ ] Stored procedures and functions (PL/pgSQL)
- [ ] Triggers
- [ ] User-defined types and domains
- [ ] Full-text search
- [ ] JSON/JSONB operations
- [ ] Database design and normalization

## Practical Tasks

- [ ] Write complex analytical queries with window functions
- [ ] Design and implement proper indexes
- [ ] Optimize slow queries using EXPLAIN
- [ ] Create stored procedures for business logic
- [ ] Implement triggers for audit logging
- [ ] Build full-text search functionality
- [ ] Work with JSON data in PostgreSQL
- [ ] Design normalized database schemas
- [ ] Handle transactions with proper isolation
- [ ] Create and use materialized views

## Projects

- [ ] Build an analytics dashboard with complex queries
- [ ] Optimize a slow database with indexing and query tuning
- [ ] Implement full-text search for a content platform

Once you can confidently check off these items, you are ready for the Green Belt!
