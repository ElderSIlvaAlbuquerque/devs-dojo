# Ruby on Rails Green Belt Checklist

## Core Concepts

- [ ] Performance optimization and caching
- [ ] Action Cable for WebSockets
- [ ] Advanced Active Record (STI, polymorphic, delegated types)
- [ ] Service objects and design patterns
- [ ] GraphQL APIs
- [ ] Hotwire (Turbo and Stimulus)
- [ ] Deployment and DevOps
- [ ] Database optimization (indexing, query performance)
- [ ] Monitoring and logging
- [ ] Scaling Rails applications
- [ ] Security hardening
- [ ] API rate limiting and throttling

## Practical Tasks

- [ ] Implement caching strategies (fragment, Russian doll)
- [ ] Build real-time features with Action Cable
- [ ] Create service objects for business logic
- [ ] Optimize database queries
- [ ] Deploy Rails app with CI/CD
- [ ] Implement comprehensive monitoring
- [ ] Build GraphQL API
- [ ] Use Hotwire for SPA-like experience
- [ ] Scale application horizontally
- [ ] Harden security posture

## Projects

- [ ] Build real-time chat application
- [ ] Create high-performance API with caching
- [ ] Implement scalable multi-tenant SaaS

Once you can confidently check off these items, you are ready for the Blue Belt!
