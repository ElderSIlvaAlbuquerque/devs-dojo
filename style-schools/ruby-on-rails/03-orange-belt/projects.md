## Project 1: Blog Platform with Authentication

**Time**: 2-3 weeks  
**Difficulty**: ⭐⭐⭐ Intermediate  
**Focus**: CRUD, authentication, authorization

### Requirements

**Features**:
- [ ] User authentication (Devise)
- [ ] Create, edit, delete posts
- [ ] Categories and tags
- [ ] Comments on posts
- [ ] User profiles
- [ ] Search functionality
- [ ] Markdown support
- [ ] Authorization (own posts only)

**Technical**:
- [ ] RESTful routes
- [ ] Active Record associations
- [ ] Background jobs for emails
- [ ] Image uploads (Active Storage)
- [ ] Pagination
- [ ] AJAX for comments
- [ ] RSpec test coverage >80%

### Deliverables
- [ ] Working application
- [ ] Deployed to Heroku/Render
- [ ] Test suite
- [ ] README with setup

### Evaluation Criteria
- **Functionality (35%)**: All features working
- **Code Quality (25%)**: Rails conventions
- **Testing (20%)**: Comprehensive tests
- **UX (10%)**: User-friendly interface
- **Documentation (10%)**: Clear README
