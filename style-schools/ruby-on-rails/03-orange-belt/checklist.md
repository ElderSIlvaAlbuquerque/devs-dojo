# Ruby on Rails Orange Belt Checklist

## Core Concepts

- [ ] Advanced Active Record (associations, validations, callbacks)
- [ ] RESTful routing and resources
- [ ] Authentication (Devise or custom)
- [ ] Authorization (Pundit or CanCanCan)
- [ ] Background jobs (Sidekiq, Active Job)
- [ ] Action Mailer for emails
- [ ] File uploads (Active Storage)
- [ ] Testing with RSpec
- [ ] Database migrations and schema design
- [ ] Asset pipeline and Webpacker
- [ ] API development (JSON responses)
- [ ] Security best practices (CSRF, SQL injection, XSS)

## Practical Tasks

- [ ] Build complete CRUD application with authentication
- [ ] Implement complex Active Record queries
- [ ] Create background job workers
- [ ] Send transactional emails
- [ ] Handle file uploads with validations
- [ ] Write comprehensive RSpec tests
- [ ] Build RESTful API endpoints
- [ ] Implement authorization policies
- [ ] Design and migrate database schema
- [ ] Secure application against common vulnerabilities

## Projects

- [ ] Build a blog platform with authentication and authorization
- [ ] Create an API-driven application
- [ ] Implement a marketplace with payments

Once you can confidently check off these items, you are ready for the Green Belt!
