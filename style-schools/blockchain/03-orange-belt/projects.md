# ⛓️ Blockchain Systems: Orange Belt Projects

Choose **one** project as your capstone.

---

## Project 1: ERC-20 Token Contract

Build a standard token contract.

**Requirements**:
- Implement ERC-20 standard
- Transfer, approve, transferFrom functions
- Supply management
- Test contract on testnet
- Audit for vulnerabilities

---

## Project 2: Simple NFT Contract

Build an NFT contract (ERC-721).

**Requirements**:
- Mint NFTs
- Transfer ownership
- Metadata support
- Enumeration support
- Deploy to testnet

---

## Project 3: Basic DeFi Protocol

Build a simple lending protocol or DEX.

**Requirements**:
- Accept deposits
- Track balances
- Enable withdrawals
- Calculate interest or fees
- Secure against common attacks

---

Choose one, build it, audit it.
