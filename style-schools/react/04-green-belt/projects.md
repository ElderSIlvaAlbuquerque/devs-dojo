**Time**: 4-5 weeks  
**Difficulty**: ⭐⭐⭐⭐ Advanced  
**Focus**: SSR, TypeScript, production features

### Requirements

**Features**:
- [ ] Server-side rendered product pages
- [ ] Static generation for landing pages
- [ ] Full-text search
- [ ] Advanced filtering and sorting
- [ ] User authentication (NextAuth)
- [ ] Admin dashboard
- [ ] Payment integration (Stripe)
- [ ] Order management

**Technical**:
- [ ] TypeScript throughout
- [ ] ISR for product pages
- [ ] API routes for backend
- [ ] Image optimization
- [ ] SEO optimization
- [ ] Performance optimization
- [ ] Accessibility (WCAG AA)

**Testing**:
- [ ] Unit tests
- [ ] Integration tests
- [ ] E2E tests with Playwright
- [ ] Accessibility tests

### Deliverables
- [ ] Production-deployed application
- [ ] Comprehensive test suite
- [ ] Performance audit results
- [ ] Documentation

### Evaluation Criteria
- **Architecture (30%)**: Well-designed Next.js app
- **Features (25%)**: Complete functionality
- **Performance (20%)**: Optimized and fast
- **Quality (15%)**: Tests and accessibility
- **Documentation (10%)**: Clear docs

## Project 3: Real-Time Collaboration App

**Time**: 4-5 weeks  
**Difficulty**: ⭐⭐⭐⭐⭐ Expert  
**Focus**: Real-time sync, complex state, performance

### Requirements

**Features**:
- [ ] Real-time collaborative editing
- [ ] Presence indicators
- [ ] User cursors and selections
- [ ] Conflict resolution
- [ ] Offline support
- [ ] Version history
- [ ] Comments and threads

**Technical**:
- [ ] WebSocket or WebRTC
- [ ] Operational transform or CRDT
- [ ] Optimistic updates
- [ ] State reconciliation
- [ ] IndexedDB for offline
- [ ] Service worker

**Performance**:
- [ ] Handle 50+ simultaneous users
- [ ] <100ms latency for updates
- [ ] Efficient diff algorithms
- [ ] Memory leak prevention

### Deliverables
- [ ] Working application
- [ ] Load testing results
- [ ] Architecture documentation
- [ ] Demo video

### Evaluation Criteria
- **Real-time Sync (35%)**: Reliable collaboration
- **Performance (30%)**: Handles scale
- **UX (20%)**: Smooth experience
- **Architecture (15%)**: Clean implementation
