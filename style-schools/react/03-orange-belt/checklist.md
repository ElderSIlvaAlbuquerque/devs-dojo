# React Orange Belt Checklist

## Core Concepts

- [ ] React Router for navigation
- [ ] State management (Context API, Redux basics)
- [ ] Custom hooks
- [ ] useEffect for side effects and cleanup
- [ ] useRef for DOM access and mutable values
- [ ] useMemo and useCallback for optimization
- [ ] Form handling and validation
- [ ] API integration with fetch/axios
- [ ] Error boundaries
- [ ] Code splitting and lazy loading
- [ ] Testing with React Testing Library
- [ ] Styled-components or CSS Modules

## Practical Tasks

- [ ] Build multi-page application with React Router
- [ ] Implement global state management
- [ ] Create reusable custom hooks
- [ ] Integrate with REST API
- [ ] Handle forms with validation
- [ ] Write component tests
- [ ] Optimize performance with memoization
- [ ] Implement error handling
- [ ] Use lazy loading for routes
- [ ] Style components systematically

## Projects

- [ ] Build a full-featured SPA with routing and state management
- [ ] Create a dashboard with API integration
- [ ] Implement an e-commerce product catalog

Once you can confidently check off these items, you are ready for the Green Belt!
