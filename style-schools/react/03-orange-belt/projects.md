## Project 1: E-Commerce Product Catalog

**Time**: 2-3 weeks  
**Difficulty**: ⭐⭐⭐ Intermediate  
**Focus**: Routing, state management, API integration

### Requirements

**Features**:
- [ ] Product listing with filters and search
- [ ] Product detail pages
- [ ] Shopping cart (Context API)
- [ ] Checkout flow (multi-step form)
- [ ] Order history
- [ ] User authentication

**Technical**:
- [ ] React Router for navigation
- [ ] Context API for cart state
- [ ] Custom hooks (useFetch, useCart)
- [ ] Form validation
- [ ] API integration
- [ ] Loading and error states
- [ ] Responsive design

**Testing**:
- [ ] Component tests (>70% coverage)
- [ ] Integration tests for user flows
- [ ] E2E tests for checkout

### Deliverables
- [ ] Working application
- [ ] Test suite
- [ ] README with setup instructions
- [ ] Demo video

### Evaluation Criteria
- **Functionality (35%)**: All features working
- **Code Quality (25%)**: Clean, reusable components
- **State Management (20%)**: Proper use of Context
- **Testing (20%)**: Comprehensive tests
