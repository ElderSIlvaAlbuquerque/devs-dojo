## Project 1: CRUD API with unit tests
**Time**: 1-2 weeks
**Tech Stack**: Ruby/Go/Node/Python
**Concepts**: This project teaches the fundamentals of building a web server, handling HTTP requests, and ensuring code quality with basic tests.
**Requirements**:
- Create a simple API for a resource like "notes" or "tasks".
- Implement full CRUD (Create, Read, Update, Delete) functionality.
- Write unit tests for each endpoint.
- Include a README with instructions on how to run the API and tests.

## Project 2: Local Docker environment with API + database
**Time**: 1-2 weeks
**Tech Stack**: Docker, a programming language for a simple API
**Concepts**: This project focuses on containerization and orchestration, essential skills for modern development workflows.
**Requirements**:
- Write a `docker-compose.yml` file.
- The compose file should define at least two services: an API and a database (e.g., Postgres, MySQL).
- The API should be able to connect to the database.
- Include a README with instructions on how to start the environment with `docker-compose up`.

## Project 3: Simple web page (React) connecting to API
**Time**: 1-2 weeks
**Tech Stack**: React, any backend API
**Concepts**: This project introduces frontend development and client-server interaction.
**Requirements**:
- Create a simple React application.
- The application should fetch data from an existing API (you can use a public API or one you built).
- Display the fetched data on the page.
- Include a README with instructions on how to run the React app.
